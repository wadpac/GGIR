# Creates Config File based on variables in function GGIR environment

Only used inside
[GGIR](https://wadpac.github.io/GGIR/reference/GGIR.md). Not intended
for direct use by user.

## Usage

``` r
createConfigFile(config.parameters = c(), GGIRversion = "")
```

## Arguments

- config.parameters:

  List with all arguments used in
  [GGIR](https://wadpac.github.io/GGIR/reference/GGIR.md).

- GGIRversion:

  GGIR version mumber to be incorported in the ConfigFile.
