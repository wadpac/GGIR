# Add split names to output object in g.part4 and g.part5

Internal function to add the recording split names to output the reports
for GGIR part 4 and 5.

## Usage

``` r
addSplitNames(x)
```

## Arguments

- x:

  Output data.frame as generated inside
  [g.part4](https://wadpac.github.io/GGIR/reference/g.part4.md) and
  [g.part5](https://wadpac.github.io/GGIR/reference/g.part5.md).
