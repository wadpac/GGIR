# Extract ID from file header object

Extract ID from file header object.

## Usage

``` r
extractID(hvars, idloc, fname)
```

## Arguments

- hvars:

  Object extracted with
  [g.inspectfile](https://wadpac.github.io/GGIR/reference/g.inspectfile.md)

- idloc:

  See [GGIR](https://wadpac.github.io/GGIR/reference/GGIR.md)

- fname:

  File (base)name.
