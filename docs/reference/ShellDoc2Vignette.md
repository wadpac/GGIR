# Extract Definition from Shell Documentation

Function extracts the documentation for a given GGIR argument as
provided in the [GGIR](https://wadpac.github.io/GGIR/reference/GGIR.md)
documentation. Function not designed for direct use by package user.

## Usage

``` r
ShellDoc2Vignette(argument = "mode")
```

## Arguments

- argument:

  Character (default = "mode"). Name of the argument to extract the
  definition.

## Value

Character object with the definition of the argument.
