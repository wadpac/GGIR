# Append GGIR milestone data from neighbouring or overlapping recordings

Splits GGIR part 1 milestone data format files based on params_general
parameters `recording_split_times`, `recording_split_timeformat`, and
`recording_split_times`.

## Usage

``` r
splitRecords(metadatadir, params_general = NULL)
```

## Arguments

- metadatadir:

  See [g.part2](https://wadpac.github.io/GGIR/reference/g.part2.md)

- params_general:

  See [GGIR](https://wadpac.github.io/GGIR/reference/GGIR.md)
