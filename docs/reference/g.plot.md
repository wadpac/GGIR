# function to generate a plot for quality check purposes

Function takes meta-data as generated by
[g.getmeta](https://wadpac.github.io/GGIR/reference/g.getmeta.md) and
[g.impute](https://wadpac.github.io/GGIR/reference/g.impute.md) to
create a visual representation of imputed time periods

## Usage

``` r
g.plot(IMP, M, I, durplot)
```

## Arguments

- IMP:

  output from
  [g.impute](https://wadpac.github.io/GGIR/reference/g.impute.md)

- M:

  output from
  [g.getmeta](https://wadpac.github.io/GGIR/reference/g.getmeta.md)

- I:

  output from
  [g.inspectfile](https://wadpac.github.io/GGIR/reference/g.inspectfile.md)

- durplot:

  number of days to plot

## Value

function only produces a plot, no values
